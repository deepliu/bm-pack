# Changelog

## 0.2.0
- 新增无旋转 3D 几何校验，修复体积快检误判为可装的问题。
- 优化求解搜索：`geometry_check=True` 时只接受真实可装候选，并允许在可行箱数范围内继续搜索。
- 重构求解器内部结构：抽出状态、几何引擎、搜索评分和优化阶段模块。
- 强化箱型替换与数量整形，后处理必须保持几何可行。
- 增加真实样例与几何回归测试。

## 0.1.2
- 更新求解流程：SKU 聚合优先、可放宽 SKU 种类限制。
- 新增箱型替换优化：在约束满足时优先更小体积箱型。
- 新增数量整形优化：每箱每个 SKU 尽量为 5/10 的整数倍。
- 文档同步更新（AGENTS.md / README.md / problem_brief.md）。
